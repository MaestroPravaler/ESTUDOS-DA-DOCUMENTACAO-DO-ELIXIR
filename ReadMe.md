### [22. PROJETOS PRÁTICOS]()
Criação de um projeto em Elixir
```
mix new "nomedoprojeto"
```
Compilação do projeto
```
mix compile
```
Execução dos Textes
```
mix text
```
Compilar e rodar o projeto dentro do iex()>

No diretório do projeto Rodar o comando:
```
iex -S mix
```
